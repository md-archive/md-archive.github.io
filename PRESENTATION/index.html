Infraestructura

vSphere

VMware vSphere es la plataforma de virtualización líder del sector para construir infraestructuras de cloud. Permite a los usuarios ejecutar aplicaciones críticas para el negocio con confianza y responder con mayor rapidez a las necesidades empresariales.

vMotion

vMotion es una migración dinámica de un servidor a otro sin tiempo de inactividad para cargas de trabajo. Durante la migración de cargas de trabajo, la aplicación sigue ejecutándose y los usuarios pueden acceder en todo momento a los sistemas que necesitan. De este modo se mantiene un nivel elevado de productividad.

¿Qué implica?

vMotion en un entorno sin almacenamiento compartido está sujeto a los siguientes requisitos y limitaciones: Los hosts deben tener licencia para vMotion. Los hosts deben ejecutar ESXi 5.1 o una versión posterior. Los hosts deben cumplir con el requisito de redes para vMotion.

¿Cómo funciona?

Cuando se migran las máquinas virtuales con vMotion y elige cambiar solo el host, el estado completo de la máquina virtual se mueve al nuevo host. El disco virtual asociado permanece en la misma ubicación en el almacenamiento que se debe compartir entre los dos hosts.

                    # Value Management Tool = VMT
                    def VMT(_p_ruta_fichero, _text_match,_text_sustituto, _name_service):
                        # opening the file in read mode
                        file = open(_p_ruta_fichero, "r")
                        replacement = ""
                        # using the for loop
                        for line in file:
                            line = line.strip()
                            changes = line.replace(_text_match, _text_sustituto)
                            replacement = replacement + changes + "\n"
                        file.close()
                        # opening the file in write mode
                        fout = open(_p_ruta_fichero, "w")
                        fout.write(replacement)
                        fout.close()
                    
                        _restart_service = "service [] restart".format(_name_service)
                        OS_SCE(_restart_service)
